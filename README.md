# TransitLens ML Core

## Overview

TransitLens ML Core is responsible for all machine learning functionality within the TransitLens ecosystem.

It consumes processed light curves generated by the `transitlens-data-pipeline` repository and produces exoplanet transit predictions.

This repository owns the complete ML lifecycle including dataset loading, model training, evaluation, inference, model versioning, and explainability.

No FITS processing or frontend functionality should exist in this repository.

---

# Responsibilities

- Load processed datasets
- Dataset validation
- Dataset splitting
- Model training
- Model evaluation
- Hyperparameter configuration
- Model checkpointing
- Model inference
- Confidence estimation
- Explainability
- Experiment tracking
- Model export
